# משימת רשימת משימות ב-React

פרויקט Todo List פשוט ב-React.

## מה יש בפרויקט

- הצגת רשימת משימות
- הוספת משימה חדשה
- מניעה מהוספת משימה ריקה
- מחיקת משימה
- סימון משימה כבוצעה
- קו חוצה למשימה שבוצעה
- סינון לפי הכל / פעילות / בוצעו
- עריכת משימה קיימת
- שמירה ב-localStorage אחרי רענון הדף

## הרצה

```bash
npm install
npm run dev
```

אחר כך נכנסים לכתובת שמופיעה בטרמינל.

## הסבר קצר

השתמשתי ב-useState בשביל לשמור את רשימת המשימות ואת הטקסט שהמשתמש מקליד.
כל משימה היא אובייקט עם id, text, isCompleted.

הוספה מתבצעת על ידי יצירת אובייקט חדש והוספה שלו למערך.
מחיקה מתבצעת עם filter.
סימון כבוצע ועריכה מתבצעים עם map.
השמירה בדפדפן מתבצעת בעזרת localStorage ו-useEffect.
